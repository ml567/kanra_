# Kanra

A frontend web travel based application built with **React** and **Vite** 

---
## Tech stack

React + Vite
Tailwind + MUI

## Setup instructions

### 1. clone the repo

```bash
git clone https://github.com/ml567/kanra_.git
cd kanra_
```
install dependencies

```bash
npm install
```

Run the server
```bash
npm run dev
